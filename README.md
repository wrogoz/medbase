# MEDBASE: "NFZ earliest possible date" http://medbase.netlify.com

# Technologies: 
- ES6
- Typescript
- React
- React Router
- Mobx
- Styled-components
- Axios
- Jest
- Enzyme



# Features:

- search a earliest possible date to a doctor (which has contract with NFZ) by specialization, district and city.

# How to start:
First:
npm install

Then:
npm start
